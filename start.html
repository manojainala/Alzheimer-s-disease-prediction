Alzhiemers Disease Prediction System

About This System

Alzhiemers Disease prediction system is designed for three main functionalities namely Train, Predict, Visualize.

Functionality number 1 is to train the system to predict or visualize the level of disease by uploading a data set. CSV (comma spaced values) is the only format accepted by the system, for updating the data set. This system is trained using the LightGBM classification algorithm.

Functionality number 2 of the system is to predict the level of Alzheimer's Disease of a person by giving the certain inputs like race, sex, smoking, drinking, heart attack and some more feature input values are given for the system to predict the Level of Disease. First, users need to select predict option from the home page and then gets redirected to the prediction page and asks to fill the mandatory values.

Functionality number 3 of the system is to visualize the level of Disease for the whole data depending on their level of impairments. First, users needs to select the SEX, CDR score, MMSE score and have to click on visualize button, then the result is obtained on the same screen showing that percentage of the total count of the people on the vertical axis and combination of all the three features i.e., combination sex, cdr score values and mmse score values on the horizontal axis.
